# theos-helloworld
theos helloworld출력해보기


## theos 설치하기

```
brew install dpkg ldid
git clone --recursive https://github.com/theos/theos.git ~/theos

```

## 환경설정

- macbook에서 실행

```
echo "export THEOS=~/theos" >> ~/.zshrc
source ~/.zshrc

```
